## 19단계: 카카오톡 알림 시스템 (예정 - 사전 준비 필요)
> 견적 의뢰/제출, 가입, 공지 등을 카카오톡으로 자동 발송

### ⚠️ 코딩 전 대표님이 먼저 준비할 사전 절차
- [ ] 1. 카카오톡 비즈니스 채널 개설 + 비즈니스 인증
  - 멀티테크니션스 명의로 채널 생성 (카카오비즈니스 파트너센터)
  - 사업자등록증 제출 (주민번호 뒷자리/개인정보 마스킹 필수 - 반려 1순위)
  - 채널 프로필에 고객센터 연락처 등록 → '발신 프로필 키' 생성됨 (필수)
- [ ] 2. 공식 딜러사 계약 (비즈고/인포뱅크 등) → API 키 발급
  - 알림톡은 카카오 공식 딜러사를 통해서만 발송 가능
- [ ] 3. 알림톡 템플릿 등록 + 카카오 심사 (영업일 2~3일 소요)

### 발송 시나리오 (정보성/광고성 구분 중요)
**✅ 알림톡 가능 (정보성 - 고객 행동에 대한 피드백)**
- [ ] 고객이 견적 의뢰 → 해당(또는 지정) 파트너에게 "새 견적 요청" 알림
- [ ] 파트너가 견적 제출 → 고객에게 "견적 도착" 알림
- [ ] 회원가입 완료 → "가입 완료 안내" (※ "축하" 표현은 광고성으로 반려 위험, "완료 안내"로)
- [ ] 파트너 승인 완료 → "파트너 가입 승인 안내"
- [ ] 포인트 지급/만료 임박 → 안내 (정보성 범위 확인 필요)

**❌ 알림톡 불가 (광고성 - 친구톡/브랜드메시지 영역)**
- [ ] 이벤트/프로모션/공지 발송 → 친구톡 또는 브랜드메시지
  - 채널 친구 추가 + 마케팅 수신동의 필요 (별도 구조)
  - 우리가 이미 받는 "마케팅 정보 수신 동의"와 연계 가능

### 비용 참고
